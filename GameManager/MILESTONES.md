## Milestone 2: Version Checker

**Status:** IN PROGRESS
**Goal:** Know at a glance which projects have outdated dependencies

### Deliverables

- [x] "Check Versions" button in dashboard header
- [x] Query npm registry API for latest versions of key packages:
  - Venus SDK (`@series-inc/venus-sdk`)
  - Phaser (`phaser`)
  - Three.js (`three`)
  - Vite (`vite`)
  - React (`react`)
  - TypeScript (`typescript`)
- [x] Read each project's `package.json` to get current installed versions
- [x] Version comparison display per project (current vs latest, color-coded)
- [x] "Outdated" badge on project cards when dependencies are behind
- [x] Expandable version details in project modal
- [x] Cache results locally to avoid repeated API calls (localStorage, 24hr TTL)
- [x] `scanner.py --versions` generates `versions.json` snapshot for dashboard

### Technical Notes

- npm registry API: `https://registry.npmjs.org/{package}/latest`
- No authentication needed for public packages
- Venus CLI version check: `venus --version` (local binary)
- `versions.json` generated by `scanner.py --versions` — run before checking versions in dashboard
